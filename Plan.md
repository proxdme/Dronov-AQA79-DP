# План автоматизации тестирования веб-сервиса покупки туров
## 1. Перечень автоматизируемых сценариев:
### Позитивные сценарии:
#### ***1.1 Успешная покупка тура с оплатой по дебетовой карте***
##### Пользователь вводит корректные данные дебетовой карты:
- Ввод номер карты из списка 4444,
- Вввод месяца действующая по сроку - например 05,
- Ввод года действующая по сроку - например 24,
- Ввод владелеца карты - введены Имя и Фамилия через пробел на латинице например Ivan Ivanov,
- Ввод кода CVC из цифр в формате ХХХ например 123.
- Нажать кнопку "Продолжить".
  
Уведомление об успешномо совершении платежа.\
Подтверждение сохранения информации о платеже в СУБД.

Уведомление об успешном совершении платежа через кредитный сервис.\
Подтверждение сохранения информации о платеже в СУБД.
